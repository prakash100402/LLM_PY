# TOM

A simple Bigram Language Model built with Python and PyTorch by following the freeCodeCamp tutorial **"Create a Large Language Model from Scratch with Python"**.

This project demonstrates the fundamentals of training a Bigram Language Model and generating text from it.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/prakash100402/LLM_PY.git
cd LLM_PY
```

### 2. Create a virtual environment (Recommended)

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**macOS/Linux**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install the required libraries

```bash
pip install torch jupyter notebook numpy matplotlib
```

> Install any additional packages if a notebook requests them.

### 4. Start Jupyter Notebook

```bash
jupyter notebook
```

or

```bash
jupyter lab
```

### 5. Open and run the notebooks

Run the notebooks in order to train the model and generate text.
